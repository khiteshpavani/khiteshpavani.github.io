Instructions to run the Training, Testing and GUI:


Dependencies
- Python 3
- Tkinter
- Keras
- Tensorflow-gpu
- Numpy
- Panda
- sklearn

Instructions
- 1. In order to perform training, unzip the file, download the 42 GB dataset from the Kaggle site and unzip the contents into the folder data.
- 2. To start the training process run: python train.py
- 2. To start the testing use the test.py file. (python test.py)
- 3. Run the following command in command prompt:
    python diagnose_xray.py
- 4. To use the GUI run the following command in command prompt:
- 5.     python gui_diagnose_xray.py
- 6. A dialog box opens up with instructions written
- 7. Use the “Load X-ray” button to select the X-ray image file to perform diagnosis on. I am providing two sample x-ray images for trial.
- 8. After this press the “Diagnose” button to find the result of the diagnosis.
