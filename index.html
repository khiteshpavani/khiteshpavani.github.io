Used different image recognition algorithms such as Densenet121, Densenet169, InceptionV3, Squeeze-and-excite technique for diagnosing x-rays.

Data set: NIH Chest X-rays (Kaggle)

Dataset: NIH Chest Dataset Kaggel Link

Instructions to run the Training, Testing and GUI:


NOTE: The package contains a small sample test data set of 64 images which can be tested using the already trained model. In order to do the training the original data set can be downloaded from the Kaggle Site mentioned above
Dependencies
- Create Conda environment file using the environment.yml file provided or install the dependencies
   Command : conda env create -f environment.yml
- Python 3
- Tkinter
- Keras
- Tensorflow-gpu
- Numpy
- Panda
- sklearn
- keraspreprocessing=1.0.5 *specifically

Instructions
*NOTE: In order to perform testing on a small sample data set, simply run "python test.py"
- 1. In order to perform training, unzip the file, download the 42 GB dataset from the Kaggle site and unzip the contents into the folder data.
- 2. To start the training process run: python train.py
- 2. To start the testing use the test.py file. (python test.py)
- 3. Run the following command in command prompt:
    python diagnose_xray.py
- 4. For running the GUI Standalone the following files are required:
  a)diagnose_xray
  b)True_Labels.csv
  c)weights.h5
  d)x_rays folder containing random x-rays
- 5. To use the GUI run the following command in command prompt:
- 6.     python gui_diagnose_xray.py
- 7. A dialog box opens up with instructions written
- 8. Use the “Load X-ray” button to select the X-ray image file to perform diagnosis on. I am providing two sample x-ray images for trial.
- 9. After this press the “Diagnose” button to find the result of the diagnosis.
